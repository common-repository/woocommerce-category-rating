== Description ==

This plugin calculates the average rating of a Woocommerce product category based on the reviews added to each individual product it contains. The rating is added in Schema.org format and it will create Rich Snippets in Google for each category that contains reviewed products.

== Installation ==


1. Upload and extract the zip file downloaded to the /wp-content/plugins/ directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= How do I add some rating text? =

Go to the Settings page of the plugin (called Category Ratings) and just add your custom text in there.
